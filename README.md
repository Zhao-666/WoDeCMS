# WoDeCMS

微信小程序做的后台管理程序

# 使用的图表组件
https://github.com/xiaolin3303/wx-charts

# 功能
* 登陆验证
* 加载中loading图
* 切换图表类型
* 可滑动视图
* 悬浮球
* 屏蔽下层界面，弹出选项卡
* 个人选项页
